NovaRetail: Análisis de Comportamiento del Consumidor y Optimización de Ventas
📊 Descripción del Proyecto
Este proyecto consiste en un análisis exhaustivo de los datos de NovaRetail, con el objetivo de identificar patrones de compra, segmentar clientes y proponer estrategias basadas en datos para mejorar la retención y el ticket promedio.

El notebook incluye desde la limpieza de datos (EDA) hasta la validación de hipótesis y el planteamiento de modelos predictivos, habiendo sido revisado y aprobado con comentarios positivos sobre la profundidad del análisis.

🛠️ Tecnologías Utilizadas
Lenguaje: Python 3.x

Librerías Principales:

Pandas y NumPy: Manipulación y limpieza de datos.

Matplotlib y Seaborn: Visualización de datos y análisis estadístico.

SciPy: Pruebas de hipótesis.

Entorno: Jupyter Notebook / Google Colab.

🚀 Características Principales
Limpieza de Datos: Tratamiento de valores nulos, duplicados y outliers para asegurar la integridad del análisis.

Análisis Exploratorio (EDA): Identificación de las categorías de productos más vendidos y periodos de mayor actividad.

Segmentación de Clientes: Análisis de comportamiento para diferenciar grupos de consumidores.

Pruebas de Hipótesis: Validación estadística de suposiciones de negocio para la toma de decisiones informada.

📈 Hallazgos Clave
Segmentación: Se identificaron nichos específicos que responden mejor a ciertas promociones.

Validación: Se realizaron pruebas de hipótesis extra que permitieron refinar las métricas de éxito del proyecto.

Propuestas a Futuro: Creación de modelos predictivos para estimar variables críticas de venta.
